# Анализатор access.log файлов

Этот скрипт анализирует access.log файлы веб-сервера и собирает статистику по запросам.

## Требования

- Python 3.6+
- Нет внешних зависимостей

## Формат логов

Скрипт ожидает логи в следующем формате: %h - - %t "%r" %s %b "%{Referer}" "%{User-Agent}" %d

Где:
- %h - IP-адрес удаленного хоста
- %t - время получения запроса
- %r - тип запроса, его содержимое и версия протокола
- %s - код состояния HTTP
- %b - количество отданных сервером байт
- %{Referer} - URL-источник запроса
- %{User-Agent} - HTTP-заголовок, содержащий информацию о запросе
- %d - длительность запроса в миллисекундах
